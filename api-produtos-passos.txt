#1 Configuração inicial
npm init -y === Inicia o projeto node
npm install express === Instala o expressjs biblioteca back-end
npm install nodemon -D === Instala o nodemon para atualizar o projeto em tempo real o -D é porque ele é apenas uma dependência de desenvolvimento, não vai para o projeto final.
npm install dotenv === Para acessar o arquivo .env com as informações do banco de dados.
npx eslint --init === Padroniza a escrita do projeto para deixar mais organizado.
